Element Definitions

<h1></h1>...<h6></h6>
Headings: displays different element sizes (h1 being the largest, h6 being the smallest)
<p></p>
Paragraphs: used to group text content together as a whole in the html
<b></b>
Bold : used to make the text in the paragraph appear bold
<i></i>
Italic: used to italicize text in the paragraph
<sup></sup>
Superscript: used to implement suffixes of dates to text
<sub></sub>
Subscript: used to implement mathematical powers to text
<br></br>
Line Breaks: used to add breaks inside paragraphs
<hr></hr>
Horizontal Rules: used to add horizontal rulers that separates lines of text in the html
<strong></strong>
Strong :used to express strong importances in text
<em></em>
Emphasis: used to express subtle changes in the meaning of a sentence in a paragraph
<blockquote></blockquote>
used to wrap a paragraph of text in quote marks
<q></q>
Quotations: used to wrap single lines of text in quote marks
<abbr></abbr>
Abbreviations and Acronyms: used to display full details of abbreviated text when highlighted
<cite></cite>
Citations: used to indicate the location of citations in the html
<dfn></dfn>
Definitions: used to define new inputted terms in the html
<address></address>
Address: used to contain personal contact details in the html
<ins></ins>
Insert: used to show the content that was inserted into the html
<del></del>
Delete: used to show the deleted content in the html
<s></s>
used to show content that may be inaccurate, but should not be deleted from the html
<ol></ol>
Ordered List: used to create an organized, numbered list in the html
<li></li>
List Item: used to identify each item in the list
<ul></ul>
Unordered List: used to create a random list with bullet points in the html
<dl></dl>
Definition List: used when creating terms and definitions
<dt></dt>
Definition Term: used to contain the term being defined in the list
<dd></dd>
used to contain the definition of the term
